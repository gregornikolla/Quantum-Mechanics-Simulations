# Quantum Mechanics Simulations ⚛️

## Overview
This repository contains numerical simulations and interactive Jupyter Notebooks exploring the fundamental concepts of Quantum Mechanics. The project serves as a computational companion to my study of *Introduction to Quantum Mechanics* by David J. Griffiths. 

The goal is to translate theoretical physics concepts—such as the Schrödinger equation, wave function collapse, and probability densities—into functional, visual Python code.

## Project Structure
* `notebooks/`: Interactive Jupyter Notebooks combining theoretical explanations (Markdown/LaTeX) with executable Python code and visualizations.
* `src/`: Core Python scripts and functions containing the mathematical logic and numerical methods.

## Technologies Used
* **Language:** Python 3
* **Libraries:** NumPy, Matplotlib
* **Environment:** Jupyter Notebooks
